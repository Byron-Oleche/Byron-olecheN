# CCNA Packet Tracer Labs

This repository contains networking labs created while studying for the Cisco Certified Network Associate (CCNA).

The goal of this repository is to document my hands-on practice with network configuration, troubleshooting, and design using Cisco Packet Tracer.

## Skills Practiced

- IP Addressing and Subnetting
- VLAN Configuration
- Inter-VLAN Routing
- Static Routing
- OSPF
- Network Troubleshooting

## Tools Used
- Cisco Packet Tracer
- Wireshark – Network packet analysis and protocol inspection
- Tera Term – Remote terminal access via SSH, Telnet, and serial console
- GitHub for version control and project documentation
  
## Lab Categories

### Subnetting Labs
Practice dividing networks into smaller subnets and assigning IP addresses.

### VLAN Labs
Configuring VLANs and trunk ports on switches.

### Routing Labs
Configuring static routes and OSPF between routers.

### Troubleshooting Labs
Diagnosing and fixing network connectivity problems.

## Purpose

This repository serves as a portfolio of networking labs demonstrating practical experience with routing, switching, subnetting, and network troubleshooting using Cisco Packet Tracer.

The goal is to showcase hands-on networking skills and continuous learning in network infrastructure and operations.
